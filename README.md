# Práctica 9 – App de Entrenamiento con IA (Google Antigravity)

Construcción de una aplicación móvil de entrenamiento por etapas, implementando la interfaz visual y la lógica de negocio mediante el uso de herramientas de Inteligencia Artificial (Google Antigravity).

## Descripción del Proyecto
Esta aplicación permite a los usuarios llevar un control detallado de sus rutinas de gimnasio. Siguiendo un diseño "Dark Athletic" moderno y funcional, la app facilita el registro de series, pesos y repeticiones, pasando de ser una simple interfaz visual a un sistema con persistencia de datos local.

## Funcionalidades Principales
* **Registro de Entrenamientos:** Captura dinámica de ejercicios, series, repeticiones y peso[cite: 1].
* **Temporizador de Descanso:** Cronómetro integrado con opciones predefinidas y personalizadas para el descanso entre series[cite: 1].
* **Historial y Biblioteca:** Consulta de entrenamientos previos y catálogo de ejercicios categorizados[cite: 1].
* **Seguimiento de Progreso:** Visualización de la evolución física y medidas corporales del usuario[cite: 1].

## Tecnologías y Herramientas
* **Framework:** React Native con Expo
* **Generación de Código (IA):** Google Antigravity mediante prompting iterativo
* **Almacenamiento Local:** AsyncStorage
* **Control de Versiones:** Git y GitHub
